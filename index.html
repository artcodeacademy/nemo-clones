Nemo Clones

In this project we will learn how to use clones creating a program to clone Nemo. Click in the example to see the final result:

Click here

1) Create a new project, choose a new backdrop and an object. If you want you can use the same ones from the example.

2) When it starts make Nemo think "I am Nemo" and 2 seconds later make it go to a random position.

3) Make sure to repeat the actions on step 2 forever, and everytime you press the green flag make Nemo start on the same position.

4) When you click Nemo play a short sound and create a clone.

Hint: Look through the control blocks to find the clone blocks.

5) When it starts as a clone make the Nemo Clone glide to a random position.

6) Now make each new clone disappear after 6 seconds, we don't want the screen to be flooded with clones.

Bonus: Add other blocks when the clones start for example, make them turn, change their size, or make them say something, you choose!

*** Important: Always remember to save your project and share it when you're done.
